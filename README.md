# Elctricity-consumption-analysis
An interactive data visualization project analyzing electricity consumption patterns across regions and states during 2019–2020 using Tableau.

## Project Overview
This project analyzes electricity consumption patterns across different regions during 2019–2020. 
The objective of this analysis is to understand how electricity usage varies across states and 
identify trends in power consumption.
The dashboard was created using Tableau to transform raw electricity data into interactive 
visualizations that provide clear and meaningful insights.

## Objectives
- Analyze electricity consumption trends
- Compare electricity usage across states
- Understand regional electricity demand
- Present insights using interactive dashboards

## Tools & Technologies
- Tableau Public
- CSV Dataset
- Data Visualization Techniques

## Dashboard Features
- Monthly electricity consumption trends
- State-wise electricity usage comparison
- Regional electricity consumption analysis
- Interactive filters for region selection

## Live Dashboard

View the interactive dashboard here:

## Insights
- Electricity consumption varies significantly across states.
- Industrial and highly populated regions show higher electricity demand.
- Monthly patterns indicate seasonal changes in electricity usage.

## Conclusion
This project demonstrates how data visualization can help analyze electricity 
consumption patterns effectively. Using Tableau dashboards, complex datasets 
can be transformed into meaningful insights that support better understanding 
of energy usage trends.
